# Unsupervised-Learning
Python code for my unsupervised learning course
Mental Health in Technology – Employee Segmentation
This project applies unsupervised machine learning to the OSMI 2016 Mental Health in Tech Survey dataset to segment technology-sector employees into distinct groups based on their mental health profile and workplace environment.

What the code does:

Loads the real OSMI 2016 dataset (1,433 respondents, 63 questions)
Cleans and preprocesses the data
Applies PCA for dimensionality reduction
Uses K-Means clustering to identify 4 employee segments
Generates 6 visualisation figures automatically
